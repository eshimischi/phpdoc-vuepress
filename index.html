phpdoc-vuepress

Template for generating your PHP API documentation in a pretty VuePress format

Get Started →

Document all your code

Generate documentation for all your classes, interfaces and traits sorted by namespace. Also, include your global/namespaced functions and constants.

Easy integration

Combine your API documentation with your existent VuePress pages by simply adding the 5 generated files to your sidebar configuration as a group.

Linked references

Navigate between classes/members references by just a click. Get references of inherited members

Searchable elements

Use the VuePress search in toolbar to find classes, interfaces, traits, functions and constants by name.
